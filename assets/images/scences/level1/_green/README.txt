放置「纯绿背景」素材的文件夹
================================

把用 AI 生成的【纯绿底像素风】素材直接放进这个文件夹，
然后在同目录的 manifest.json 里登记文件名（已预置好），
最后告诉我「抠图」，我会运行 tools/chromakey.py 生成透明 PNG 并接入场景 CSS。

────────────────────────────────────────
生成建议（重要）
────────────────────────────────────────
1. 背景务必用【纯色绿】，不要渐变、不要杂色。推荐色值 #00FF00（最易识别）。
   像素风硬边素材最好抠，导出 PNG（千万别导出 JPG，会引入绿幕噪点）。
2. 物体本身尽量少用纯绿。树叶/草请用偏黄绿 #5A7A2A，否则会被一起扣掉。
3. 每张图建议尺寸（与关卡场景匹配）：
   - 村口远山背景       1920 x 600  横幅
   - 畲族村口寨门        300 x 400
   - 绣花古宅外景        400 x 500
   - 畲族大戏台          300 x 400
   - 畲族祠堂外景        400 x 500
   - 后山染布晒场背景   1920 x 600  横幅
   - 前景装饰包(树/草/栅栏) 128 x 128（可拼一张）
   - 雾气纹理层          512 x 256  半透明（此层不抠，直接当叠加用）

────────────────────────────────────────
manifest.json 说明
────────────────────────────────────────
{
  "inputs": {
    "gate_bg_far.png": { "out": "gate_bg_far.png", "resize": [1920,600] },
    ...
  }
}
- out      : 输出文件名
- resize   : 可选，强制缩放到 [宽,高]（像素风用最近邻）
- threshold: 可选，扣色强度（默认 110，越大扣越多）
- soft     : 可选，边缘软过渡（默认 28）
- target_green: 可选，固定绿幕色 [r,g,b]；不填则自动检测四角
