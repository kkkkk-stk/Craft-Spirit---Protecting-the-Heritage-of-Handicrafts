## 一、整体美术风格定位

> **核心风格：新中式轻水墨风（现代中国水墨风）**  
> **艺术调性：素雅、静谧、空灵、高远，带有一丝禅意与书卷气**  
> **适用场景：国风游戏背景、插画插图、唯美古风美术设定**

### 1.1 风格定义

- **核心风格**：新中式轻水墨风（现代中国水墨风）
- **艺术调性**：素雅、静谧、空灵、高远，带有一丝禅意与书卷气
- **融入元素**：畲族/浙江非遗传统纹样（盘瓠纹、凤凰纹、山河纹、花草纹）
- **游戏适配**：场景以水墨手绘静态背景为主
  - **角色立绘**：新中式水墨手绘风格（对话界面大尺寸半身/全身图）
  - **角色精灵图**：**像素风（Pixel Art）**，用于场景中行走、待机、交互等帧动画
- **邪气表现**：淡灰黑气纹理覆盖表面（不恐怖，代表文脉枯萎），融入水墨晕染技法
- **特效规范**：仅彩线、五色柔光、细碎纹样光点，无刺眼闪光，特效边缘用水墨晕染处理

---

### 1.2 线条表现（Line Work）—— 「骨法用笔，疏密有致」

- **线稿质感**：并非干净锐利的矢量线，而是模仿**毛笔勾线（白描）** 的质感。线条带有细微的墨色深浅变化、干湿浓淡（有些许飞白和毛边）。
- **粗细与虚实**：
  - **实体/近景**（如桌椅、房屋结构、文物道具）：线条较实、较黑，结构交代清晰。
  - **远景/背景**（如远山、云雾）：基本**无硬线勾勒**，直接用色彩渲染（没骨法），或用极淡的虚线带过。

> **⛔ 硬性禁止**：禁止使用粗细均匀的死板黑线。必须使用带压感、带颗粒感（类似炭笔或毛笔）的笔刷勾缝。

---

### 1.3 色彩与光效（Color Palette & Lighting）—— 「墨分五色，暖光点睛」

#### 主色调

以**大地色系（暖灰、赭石、熟褐）** 和**水墨灰色**为主。色彩饱和度极低，呈现出纸张泛黄的古朴质感。

| 色彩角色 | 色系 | 用途 |
|----------|------|------|
| 底色 | 宣纸米白/米黄（≈ #F2EFE9） | 背景底色、界面底色 |
| 主色 | 暖灰、赭石、熟褐 | 建筑、家具、山体 |
| 辅助色 | 青瓦灰、草木绿 | 屋顶、植被、染材 |
| 点缀色 | 畲族五色（黑/青/黄/绿/红） | 刺绣纹样、丝线、图腾高光 |
| 光效 | 暖黄色微光（Warm Glow） | 油灯、CG 画面暖光、匠灵柔光 |

#### 底色质感

背景带有微微的**宣纸/水彩纸纹理（Paper Texture）**，不是纯白，而是温润的米白或米黄色（≈ #F2EFE9）。

#### 光影表现

- 整体是**散点透视和无明显强光源**的传统国画画法。
- **局部点睛光**：画面中唯一的亮色是油灯或匠灵散发的**暖黄色微光（Warm Glow）**，采用柔和的渐变晕染，打破画面的冷寂，冷暖对比非常克制。

> **⛔ 硬性禁止**：不要画强光直射的投影（如硬边缘的影子）。阴影用淡淡的湿水墨晕染代替。严禁使用高饱和度颜色。

---

### 1.4 构图与留白（Composition & White Space）—— 「虚实相生，意境深远」

- **留白艺术**：画面超过 60% 的区域为「虚」的云雾和天空，给观者留下想象空间（计白当黑）。
- **前后景深（三远法）**：
  - **近景（实）**：精细的木桌、笔筒、画卷、灯盏、文物道具。
  - **中景（过渡）**：古风建筑、屋檐、晒架。
  - **远景（虚）**：层叠的泼墨远山，通过水墨的干湿浓淡表现由近及远的透视感（空气透视）。

> **关卡应用**：
> - 绣花古宅：近景绣花台+中景室内陈设+窗外远景山雾
> - 后山晒场：近景晒架染缸+中景晒场竹架+远景畲族群山
> - 祠堂：近景石壁织机+中景长案+窗外薄雾远山

---

### 1.5 材质与笔触（Texture & Brushwork）—— 「水渍边缘，湿画晕染」

- **水彩/水墨晕染**：在山峦和阴影处，有非常明显的**水渍边缘（Wet-on-wet watercolor edge）**。
- **材质纹理**：木质结构、屋顶瓦片上保留了手绘的扫笔和干枯笔触，增加了历史感和斑驳感。

> **上色核心要求**：
> 1. 色彩过渡要像水在纸面上洇开一样自然，边缘要有水渍感。
> 2. 使用「正片叠底」或半透明图层在底纹上铺色，保留背景宣纸的纹理。
> 3. **严禁使用高饱和度颜色**。

---

### 1.6 制作规范（给美术人员的硬性要求）

| 规范项 | 要求 |
|--------|------|
| **线稿** | 禁止粗细均匀死板黑线；必须使用带压感、带颗粒感（类似炭笔或毛笔）的笔刷勾缝 |
| **上色** | 「正片叠底」或半透明图层铺色，保留宣纸纹理；色彩过渡自然水渍边缘 |
| **光影** | 不强光直射投影；阴影用湿水墨晕染代替 |
| **饱和度** | **严禁高饱和度颜色**，全程低饱和大地色+水墨灰 |
| **特效** | 粒子/光效边缘用水墨晕染处理，不应出现干净利落的矢量边缘 |
| **精灵图风格** | **角色精灵图（行走/待机/交互动画）必须采用像素风（Pixel Art）**；单帧尺寸 32×48 ~ 128×256，参考主角现有风格：`protagonist_move/` 系列 (228~297 KB/帧) |
| **立绘风格** | 对话用半身/全身立绘采用新中式水墨手绘，≥ 1024×1536；CG 用大场景水墨手绘 |
| **图标** | 文物/UI 图标 ≥ 64×64 像素风或水墨风格均可 |
| **分辨率** | 背景图 ≥ 1920×1080，立绘 ≥ 1024×1536，像素精灵帧 ≥ 32×48 |
| **格式** | 交付 PSD/AI 源文件 + PNG 透明底导出 |
| **纹理** | 所有图面保留宣纸/水彩纸纹理，不可纯白纯黑 |

### 7.2 命名规范

| 规范项 | 规则 |
|------|------|
| **文件名** | 全小写英文 + 下划线，如 `lan_apo_portrait.png` |
| **层级前缀** | L1- 表示第一关专用，GEN- 表示通用 |
| **用途后缀** | `_bg` 背景、`_portrait` 半身、`_stand` 全身 |
| **格式** | 图片统一 PNG 透明底，矢量源文件保留 AI/SVG |
| **分辨率** | 背景图 ≥ 1920×1080，立绘 ≥ 1024×1536，图标 ≥ 64×64 |
| **交付物** | 每张图片附带原始源文件（PSD/AI），便于后续调整 |
