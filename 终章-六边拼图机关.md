# 六边锦绣拼图 · 实施文档

---

## 一、基础信息

| 项目 | 内容 |
|------|------|
| **机关名称** | 六边锦绣 |
| **所属位置** | 关卡一（畲族刺绣）· 绣花古宅内 · 终极小游戏 |
| **解锁条件** | 无（三阶全部自由开放） |
| **核心玩法** | 点击旋转六边格，还原图像 |

## 二、三阶配置

| 阶段 | 六边格数 | 蜂窝环数 | 难度 | 预计时间 |
|------|----------|----------|------|----------|
| 第一阶 | 7格 | 1环 | 入门 | 2-3分钟 |
| 第二阶 | 19格 | 2环 | 中等 | 5-8分钟 |
| 第三阶 | 37格 | 3环 | 挑战 | 8-12分钟 |

> 三阶共用同一张 `last1.png`，切割密度递增。后续可替换为三张不同绣图。

## 三、操作方式

| 操作 | 方式 | 效果 |
|------|------|------|
| 顺时针旋转60° | 鼠标左键点击六边格 | 格子顺时针转一格 |
| 逆时针旋转60° | 鼠标右键点击六边格 | 格子逆时针转一格 |
| 查看原图 | 点击"查看原图"按钮 | 半透明叠加完整参考图 |
| 重新打乱 | 点击"重新打乱"按钮 | 所有格子重新随机旋转 |
| 返回选关 | 点击"返回选择"按钮 | 回到三阶选择界面 |

## 四、技术实现

### 4.1 核心方案：CSS background-position

**不切割图片**，所有六边格共用同一张 `last1.png` 作为背景：

- `background-size: 600px 520px`（缩放到棋盘尺寸）
- `background-position: -leftpx -toppx`（每格偏移到对应区域）
- `clip-path: polygon(100% 50%, 75% 100%, 25% 100%, 0% 50%, 25% 0%, 75% 0%)`（flat-top 六边形裁剪）
- `transform: rotate(Xdeg)`（旋转，背景图跟随旋转）

旋转时背景图随格子一起转，正六边形6次对称所以形状不变，只有图像内容变化。

### 4.2 文件结构

```
css/hexPuzzle.css                    ← 拼图样式
js/hexPuzzle/hexPuzzleManager.js     ← 核心逻辑模块
index.html                           ← 拼图HTML结构 + CSS引入
js/main.js                           ← 模块初始化
js/level1/level1Manager.js           ← 入口按钮绑定
```

### 4.3 蜂窝坐标系统

采用 axial 坐标 (q, r)，flat-top 六边形：

```
像素中心：
  cx = boardW/2 + 1.5 * size * q
  cy = boardH/2 + sqrt(3) * size * (r + q/2)

有效坐标：max(|q|, |r|, |q+r|) <= N
  N=1 → 7格, N=2 → 19格, N=3 → 37格
```

### 4.4 通关判定

每个六边格的正确旋转角度为 0°。初始随机旋转 1~5 次（60°~300°）。
当所有格子 `rotation === 0` 时判定通关。

## 五、游戏内入口

1. 开始游戏 → 进入关卡一（畲族刺绣）
2. 行走至绣花古宅，按 E 进入
3. 古宅内底部出现「六边锦绣 · 终极拼图」入口
4. 点击进入 → 选择阶段 → 开始拼图
